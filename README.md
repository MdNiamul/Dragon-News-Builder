# 🐉 Dragon News

**Dragon News** is a modern and responsive online news platform where users can easily read the latest news by category. The website provides a smooth and fast experience for exploring different news topics.

## 🌐 Live Preview

Local Development Link:
(https://dragon-news-50a6a.web.app/)

## 🚀 Features

* 📰 Category-based news browsing
* 📖 Read full news details
* 🔐 User authentication system
* 🔑 Social login (Google / GitHub)
* 📱 Fully responsive design
* ⚡ Fast and smooth navigation
* 🎨 Modern UI with Tailwind CSS

## 🛠️ Technologies Used

* React.js
* React Router
* Tailwind CSS
* Firebase Authentication
* JavaScript (ES6)

## 📂 Project Structure

```
src
 ┣ Components
```

## ⚙️ Installation

Clone the repository

```
git clone [https://github.com/your-username/dragon-news.git](https://github.com/MdNiamul/Dragon-News-Builder.git)
```

Go to the project folder

```
cd dragon-news
```

Install dependencies

```
npm install
```

Run the project

```
npm run dev
```

## 🔐 Authentication

This project uses **Firebase Authentication** for secure user login and registration.

Supported login methods:

* Email & Password
* Google Sign-In
* GitHub Sign-In
